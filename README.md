# 2026-1 Capstone — 주사위 전투 로그라이크

Unity 6 (`6000.3.11f1`) 기반 턴제 주사위 전투 프로토타입. 플레이어는 "도박사"로서 주사위를 굴려 적과 싸우며, 탐험-전투-보상 루프를 반복합니다.

## 게임 흐름

```
메인 메뉴 → 캐릭터 선택 → 탐험(걷기) ↔ 전투 → 보스 → 승리
                                    ↕
                                 아이템 상자
```

| 씬 | 역할 |
|---|---|
| MainMenu | 시작, 설정 |
| CharacterSelect | 캐릭터 선택 (현재 Dice만 플레이 가능) |
| GameExplore | 걷기 → 일반전투 / 아이템 상자 / 보스전 이벤트 순환 |
| GameBattle | 턴제 주사위 전투 |

## 전투 시스템

### 공격 페이즈

5개의 d6 물리 주사위를 최대 3회 굴립니다. 주사위를 클릭하면 홀드(보관), 다시 클릭하면 해제합니다. 홀드된 주사위는 다음 굴림에서 제외됩니다.

| 족보 | 데미지 | 비고 |
|---|---:|---|
| YACHT (5개 동일) | 50 | |
| Four of a Kind | 40 | |
| Large Straight (2-6) | 35 | |
| Large Straight (1-5) | 30 | |
| Full House (3+2) | 25 | |
| Small Straight (4연속) | 20 | |
| 없음 | 눈의 합 | |

- 족보 공격: 대상 100%, 나머지 적 50% 스플래시
- 일반 공격: 스플래시 없음

### 방어 페이즈 (적별 개별 처리)

적이 순서대로 공격하며, 각 적의 공격마다 플레이어가 방어 굴림을 합니다.

| 적 랭크 | 방어 굴림 | 방어 판정 |
|---|---|---|
| 1~3 (★~★★★) | 1회, 자동 판정 | 완전 일치: 100% 방어, 불일치: 피격 |
| 4~5 (★★★★~★★★★★) | 3회 (홀드 가능) | 유사도 비례 — 5개 일치 100%, 4개 80%, 3개 60%... |
| 4~5 + 족보 | 3회 (홀드 가능) | 같은 족보 성공 시 100%, 실패 시 피격 |

적 공격 데미지: `ceil(랭크 × 배율)` (반칸 단위)

### 체력 (하트)

- 5칸 (10반칸) 레드 하트로 시작
- 하트 타입: Red (일반), Black (아머), Blue (임시)
- 데미지 흡수 순서: Blue → Black → Red
- UI: ● 꽉 참 / ◐ 반칸 / ○ 빈 슬롯

## 적 시스템

| 이름 | 랭크 | HP 범위 |
|---|---:|---|
| 슬라임 | 1 | 30~40 |
| 고블린 | 2 | 18~25 |
| 박쥐 | 3 | 10~15 |
| 해골 | 2 | 22~30 |
| 보스 (어둠의 지배자) | 5 | 120 |

랭크 = 적이 굴리는 주사위 수. 4개 이상이면 족보 가능.

## 파워업

탐험 중 아이템 상자에서 획득:

| 파워업 | 효과 |
|---|---|
| OddEvenDouble | 모든 눈이 홀수 또는 짝수면 데미지 2배 |
| AllOrNothing | 족보 데미지 ÷2, 일반 데미지 ×2 |
| ReviveOnce | 치명타 1회 무효화 (1회용) |

## 프로젝트 구조

```
Assets/
  Editor/              씬 빌더 (에디터 전용, Tools 메뉴에서 실행)
  Scripts/
    Battle/            전투 컨트롤러, 데미지/방어 계산, 적 주사위, 하트 표시
    CharacterSelect/   캐릭터 선택 UI
    Dice/              YachtDie 물리 주사위, 뷰포트 인터랙션
    Explore/           탐험 컨트롤러 (이벤트 순환, 걷기)
    Game/              세션 관리, 하트 시스템, 디버그 콘솔, 파워업
    MainMenu/          메인 메뉴 UI
  Dices/               주사위 3D 프리팹
  Mobs/                몹 스프라이트, 배경 에셋
  Player/              플레이어 아이들 스프라이트
```

## 씬 빌더

씬 파일(`.unity`)은 Git에 포함하지 않습니다. 에디터 메뉴에서 빌드:

| 메뉴 | 씬 |
|---|---|
| `Tools > Build MainMenu Scene` | MainMenu |
| `Tools > Build CharacterSelect Scene` | CharacterSelect |
| `Tools > Build GameExplore Scene` | GameExploreScene |
| `Tools > Build GameBattle Scene` | GameBattleScene |

빌드할 때마다 물리 머티리얼, 렌더텍스처 등 에셋을 최신 값으로 재생성합니다.

## 디버그 콘솔

코나미 커맨드(↑↑↓↓←→←→BA)로 활성화, `Esc`로 닫기.

| 명령 | 효과 |
|---|---|
| `/setdice N N N N N` | 주사위 결과 강제 (1~6) |
| `/kill player` | 플레이어 즉사 |
| `/kill mob @a` | 전체 적 처치 |
| `/kill mob 0 1 2` | 인덱스 지정 적 처치 |
| `/help` | 명령어 목록 |
